## Задания для самостоятельной работы и зачета

### 1. Параметры поиска границ доменов

В блокноте `3.1_call_boundaries_from_hic.ipynb` поменяйте параметры:

- `resolution`
- `window`
- набор `windows`, если хотите сравнить несколько diamond score

Для каждого варианта сохраните число найденных границ и затем запустите блокнот `3_ctcf_vs_domain_boundaries.ipynb`.

Что нужно сравнить:

- сколько границ доменов детектируется при разных параметрах
- какая доля границ пересекается с CTCF пиками
- как меняется CTCF bw профиль вокруг границ
- как меняется обогащение относительно случайного контроля

На зачете можно показать несколько графиков и объяснить, почему результат зависит от разрешения Hi-C карты и размера diamond window

### 2. Промоторы, H3K27Ac, метилирование и RNAseq

Вытащите промоторы генов: `1000 bp` перед TSS (берем аннотацию с ftp genome/[chm13v2.0_main_protein_coding_chrNames.gtf.gz](https://genedev.bionet.nsc.ru/ftp/by_User/DashaPanchenko/OMICS_course_spring_2026/genome/chm13v2.0_main_protein_coding_chrNames.gtf.gz), там только белок кодирующие гены)

Важно учитывать ориентацию гена, на минус цепи TSS будет большей координатой

- для `+` цепи промотор: `TSS - 1000 bp` до `TSS`
- для `-` цепи промотор: `TSS` до `TSS + 1000 bp`

Дальше для промоторов посчитайте:

- сколько промоторов содержат H3K27Ac пики
- в каких промоторах низкое метилирование
- какой RNAseq bw сигнал в этих промоторах

Вопрос для интерпретации: согласуются ли активный H3K27Ac, низкое метилирование и высокий RNAseq сигнал?

### 3. Разбор отдельных геномных регионов в IGV

Найдите в IGV несколько регионов активных генов

Для каждого региона сравните:

- RNAseq bw сигнал
- ChiPseq bw треки
- метилирование
- A/B компартмент
- есть ли рядом CTCF пики или границы доменов

На зачете можно показать 2-3 региона и коротко объяснить, почему вы считаете ген активным и как это видно по разным данным

### 4. Дополнительная идея: сравнить активные и неактивные промоторы

Разделите промоторы на две группы по RNAseq bw сигналу:

- активные промоторы
- неактивные промоторы

Сравните между группами:

- H3K27Ac bw сигнал
- H3K9me3 bw сигнал
- метилирование
- долю A/B компартментов

Это хорошее задание, потому что оно объединяет RNAseq, ChiPseq, метилирование и Hi-C.

### 5. Дополнительная идея: CTCF пики около активных и неактивных генов

Проверьте, отличается ли расстояние до ближайшего CTCF пика для активных и неактивных генов

Можно сделать простой анализ:

- взять TSS генов
- найти ближайший CTCF пик
- построить распределение расстояний для активных и неактивных генов

Главный вопрос: видим ли мы связь между архитектурными элементами генома и активностью генов?
